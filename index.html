∞ Идеи для следующей итерации
Когда подключим Jira API, этот дашборд можно сделать полностью автоматическим:
  • Подтягивать закрытые спринты с доски.
  • Фильтровать задачи по статусам «Done / Closed / In Progress».
  • Маппить задачи на стримы по эпикам / компонентам.
  • Пересчитывать проценты и перерисовывать графики без ручного Excel.
