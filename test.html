∞ Идеи для следующей итерации
Когда подключим Jira API, этот дашборд можно сделать полностью автоматическим:
  • Подтягивать спринты по статусу closed с доски.
  • Фильтровать задачи по статусам «Closed / Done / В анализе / В работе».
  • Маппить задачи на стримы по эпикам / компонентам.
  • Пересчитывать проценты и перерисовывать эти же графики без ручного Excel.
